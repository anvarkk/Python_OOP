# Лабораторная работа №3: Наследование и полиморфизм

## Цель
Научиться создавать иерархии классов, использовать наследование, полиморфизм и интегрировать разнотипные объекты в коллекцию.

## Реализовано
- Базовый класс `Media` (абстрактный) с общими полями и методом `get_info()`.
- Дочерние классы:
  - `Book` (добавлен автор, метод `apply_discount`, переопределён `get_info`).
  - `Magazine` (добавлен номер выпуска и месяц, метод `apply_seasonal_discount`).
- Коллекция `MediaCollection` – универсальный контейнер для хранения любых объектов, наследующих `Media`. Поддерживает добавление/удаление, поиск по названию, итерацию.
- Полиморфизм демонстрируется вызовом `get_info()` для разных объектов.
- Проверка типов через `isinstance()`.

## Демонстрация
Файл `demo.py`:
- создание книг и журналов;
- добавление в коллекцию;
- полиморфный вывод информации;
- проверка типов;
- поиск по названию;
- работа с дубликатами;
- изменение цен через свойства;
- вызов специфических методов.

# Лабораторная работа №1: Класс и инкапсуляция (Python 3.x)

## Предметная область
Класс **Book** представляет книгу в библиотеке или магазине.  
Атрибуты:
- `title` (название) – строка, не пустая.
- `author` (автор) – строка, не пустая.
- `year` (год издания) – целое число от 1440 до 2025.
- `price` (цена) – положительное число (float или int).

## Реализованный класс
Файл: `src/lab01/model.py`

### Особенности реализации
- Все атрибуты закрытые (с префиксом `_`).
- Конструктор с проверкой типов и допустимых значений.
- Свойства (`@property`) для чтения всех атрибутов; для `year` и `price` реализованы сеттеры с валидацией.
- Магические методы:
  - `__str__` – удобочитаемое представление книги.
  - `__repr__` – формальное представление, пригодное для воссоздания объекта.
  - `__eq__` – сравнение книг по всем полям.
- Атрибут класса `total_books` – счетчик созданных экземпляров.
- Бизнес-методы:
  - `apply_discount(percent)` – уменьшает цену на заданный процент (0 < percent < 100).
  - `is_modern()` – возвращает `True`, если книга издана после 2000 года.

### Демонстрация
Файл: `src/lab01/demo.py`  
Демонстрирует:
- создание объектов;
- вывод через `print`;
- сравнение объектов;
- обработку некорректных данных (try/except);
- использование сеттеров с валидацией;
- доступ к атрибуту класса через класс и экземпляр;
- применение бизнес-методов.
